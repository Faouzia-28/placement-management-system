Instructions

If no backend is found:

  1. Start Docker containers: 
    docker-compose up -d
  2. Check container status: 
    docker ps
  3. Check backend logs: 
    docker-compose logs backend
  4. If containers aren't running, rebuild them: 
    docker-compose down
docker-compose up --build -d
